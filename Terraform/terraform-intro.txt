Terraform:

- Developed by Hashicorp
- Replacement for Vagrant
  - Popular tool for creating de/test/qa/stage env
  - Mostly used along with Oracle VirtualBox
- Automate your infrastructure - IAC
  - Mutable infrastructure
  - Immutable infrastructure
- Written in GoLang
  - Supports CLI
  - 

Mutable deloyments:
 - Infrastructure can be modified. Virtual machines can be updated with new packages.

Immutable deployments:
 - Infrastructure is re-created whenever there is an update for the application.
